# oric-1 atmos rebuilt

Oric Features
- EREBUS Loader by Kenneth
- 2 ROMs with 3 Versions (Oric-1, Atmos, Erebus)
- PS2 Keyboard
- K7 Emulator with an ESP32
- UART 6551
- PIA 8255
- Original I/O connectors
- 3.5 Jack Sound output
- External 5V power supply
